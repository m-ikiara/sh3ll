## Getting Started[^1]

1. Clone the repo using the following command:

```console
$ git clone https://github.com/m-ikiara/shell.git
```

2. Build the executable:

```console
$ make -B -j3
```

3. Run the executable and pass in your commands as you would in Bash:

```console
$ ./sh3ll
```
